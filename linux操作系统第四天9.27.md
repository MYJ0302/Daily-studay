## 十四、链接应用  
链接目的就是为了在系统的多个地方都可以访问这个文件  
格式：**ln [命令选项] 源文件路径 链接文件路径**  
命令选项：  
- **-s** 创建的是软链接文件  
### 1.软链接和硬链接的区别  
硬链接：硬链接就是目标文件的一个别名，当修改其中一个硬链接文件的内容时，其他所有硬链接文件都会发生改变，删除任意一个硬链接，不会影响到其他硬链接文件或源文件本身，只有所有硬链接文件和源文件都删除时，这个文件才会被真正删除  
软链接：软链接存储目标文件或目录的路径，修改某个软链接文件，其他软链接以及源文件都会发生改变，删除某个软链接文件不会对其他文件和源文件造成影响，删除或移动源文件时，其他软链接文件都无法使用。  

## 十五、查找命令  
### 1.从指定文件中查找内容：**grep**  
格式：**grep [命令选项] 查找的关键字 操作文件路径**  
命令选项：  
- **-i** 不区分大小写  
- **-n** 连行号一起显示出来  
- **-c** 总结搜索关键字的有几行  
- **-E** 可以同时搜索多个关键字  
说明：当搜索多个关键字时，多个关键字使用|进行分隔，并将其放在双引号中  
### 2.从指定目录中查找需要文件或目录 **find**  
格式：**find 需要操作目录 [命令选项] **  
常用命令选项：  
- **-name 文件名** 按文件名进行搜索  
- **-type 文件类型** 按文件类型进行搜索  
文件类型：
  - **f** 表示普通文件  
  - **d** 表示目录  
  - **l** 表示软链接文件  
  - c 表示字符设备文件  
  - b 表示块设备文件  
  - s 表示套接字文件  
  - p 表示管道文件  
- **-size 文件大小** 按照文件大小进行搜索  

## 十六、压缩和打包相关命令  
### 1.**bzip2**  
格式：**bzip2 [命令选项] 需要压缩的文件列表**  
bzip2 此时会将每个文件进行压缩，并会自动在压缩包文件后面跟上后缀：bz2 默认情况下源文件会自动消失  
命令选项：  
- **-z** 表示压缩 （默认自带，不需要自己写）  
- **-d** 表示解压缩  
- **-k** 保留源文件  
### 2.**gzip**  
格式：**gzip [命令选项] 需要压缩文件路径**  
命令选项：  
- **-n** 表示压缩率，取值范围1-9，默认6，数字越大，压缩率越高，压缩后文件越小，压缩的时间越长  
- **-d** 解压缩  
gzip压缩后文件后缀 .gz  
说明：  
gzip在创建压缩包或解压缩时，源文件自动消失   
gzip默认压缩率会比bzip2压缩率要大  
### 3.**zip&unzip**  
- 压缩：**zip**  
格式：**zip 压缩包文件（后缀zip） 需要压缩文件列表  
- 解压缩：**unzip**  
格式：**unzip -d 解压缩后到达的目录路径 需要解压缩文件**  
- 查看压缩包文件中内容（查看压缩包压缩哪些文件）  
格式：**unzip -l 压缩包文件 zip**  
### 4.**tar**  
tar既可以实现打包，也可以实现压缩  
打包：只是打包，方便归档，文件没有变小反而变大，打包目的混进方便数据传输   
格式：**tar [命令选项] 打包文件或压缩包文件 需要打包或压缩文件列表  
tar打包后 文件后缀.tar  
tar创建压缩包文件后缀 gz  
命令选项：  
- **-c** 表示创建打包  
- **-v** 显示整个打包或解包或创建压缩包或解压缩过程  
- **-f** 打包或解压缩文件  
- **-x** 解包或解压缩  
- **-z** 表示压缩 压缩后文件后缀 gz  
- **-C** 解包或解压缩到达目的地目录路径  
- **-t** 表示查看打包或压缩包文件有哪些信息  

## 十七、安装软件相关命令  
### 1.rpm安装    
使用rpm进行安装时，一定要下载好软件rpm包  
查看当前Linux系统中已经安装的软件包（rpm软件包）  
格式：**rpm -qa**  
查看是否安装某个软件包  
格式：**rpm -q 软件包名称**  
安装：**rpm -ivh 软件包全名**  
卸载：**rpm -e 软件包名称**  
升级：**rpm -U 软件包名称**  
rpm在进行软件安装时 ，若软件之间存在依赖关系，rpm无法解除依赖，必须要使用rpm时，相应依赖包也需要下载并安装  

### 2.yum安装  
yum安装时，若软件之间存在依赖关系，他会自动解除依赖，一并下载依赖包并安装  
在使用yum进行安装之前，需要进行yum配置  
linux系统上yum配置文件所在目录路径：/etc/yum.repos.d yum配置文件通常后缀：repo  
yum源搭建过程：  
（1）创建yum源文件：  
（2）编辑yum源文件：添加资源库url地址  
（3）yum常用命令：  
- **yum list**查看当前yum资源库有哪些软件安装包  
- **yum install 软件包名称** 安装指定软件  
- **yum remove 软件包名称** 卸载指定软件  
- **yum clean all** 清理yum占用缓存  
安装指定软件：yum -y install tree  
卸载指定软件：yum -y remove  
**-y** 表示在整个安装过程中，若遇到提示信息，都默认输入y表示同意  
### 3.源码安装  
一般首先下载安装包，文件后缀通常tar.gz  
使用 tar -xzf 需要解压缩gz文件，看到被安装的软件源码  
源码安装三步骤：  
（1）./condigure configure脚本文件时源码文件中所包含文件，不是用户自己创建文件  
使用该脚本指定软件安装的目录  
（2）make 将源代码编译成二进制文件  
（3）make install 开始安装指定软件  
**yum和rpm区别和联系**  
联系：都可以安装指定软件和卸载软件  
安装：  
rpm -ivh  软件包全名称  
yum install 软件包名称  
卸载：  
rpm -e 软件包名称  
yum remove 软件包名称  
区别：
rpm在安装时，需要提前下载好依赖包和被安装的软件包。并按照一定的顺序进行安装；rpm无法解除需要安装的软件之间的依赖关系  
yum在安装使用前，需要在计算机上配置yum源资源库文件，才可以使用，可以解除软件依赖关系  

## 十八、管道符 **|**  
作用：链接两个命令 可以将前一个命令中执行结果作为后面命令输入  
格式：**命令1 | 命令2**  
使用管道符结合grep  
cat aa.txt | grep hello (grep从cat aa.txt执行结果中搜索hello)  
统计当前Linux系统一共安装多少个软件包  
rpm -qa | wc -l ——>wc -l 在统计rpm -qa 执行之后结果数量  

## 十九、进程命令  
### 1.**ps命令**  
ps 用来查看系统当前运行的进程信息  
格式：**ps [命令选项]**  
命令选项：  
- **-ef** 显示系统中所有进程的列表，包括其他用户的进程，查看进程父子关系  
- **-aux** 显示详细进程信息，包括cpu和内存使用情况
### 2.结束进程 **kill**  
格式：kill [命令选项] 需要结束进程编号  
命令选项： 
- **-9** 强制关闭某个进程  
当关闭父进程时，对应子进程也会被关闭  
### 3.设置后台进程 **&**  
**sleep 时间（时间单位秒）&** 作用在当前命令行窗口中停顿指定时间  
### 4.查看后台进程信息 **jobs**  
格式：**jobs**  
### 5.将后台进程调到前台执行 **fg**  
格式：**fg %任务编号**  
### 6.将前台执行进程调到后台执行 **bg**  
格式：**bg**  
bg在使用之前，需要使用ctrl+z暂停前台进程  

## 二十、系统资源查看命令  
### 1.用来显示整个磁盘使用的情况  
格式：**df [命令选项]**  
命令选项：  
- **-h** 以方便阅读的方式进行查看  
### 2.用于显示某一个特定的目录或文件所占用磁盘情况  
格式：**du [命令选项] 需要查看目录或文件路径**  
命令选项：  
- **-h** 以方便阅读的方式进行查看  
### 3.查看内存使用情况  
格式：**free [命令选项]**  
命令选项：  
- **-h** 以方便阅读的方式进行查看  
### 4.实时查看系统资源情况  
格式：**top [命令选项]**  
命令选项：  
- **-d 数字** 设置刷新频率  
- **-n数字** 设置结果次数 数字表示实时获取几次刷新结果，若不设置top会一直执行，知道ctrl+c才会停下
